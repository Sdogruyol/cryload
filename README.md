# cryload

HTTP benchmarking tool writen in [Crystal](http:://www.crystal-lang.org)

## Installation

```crystal
  git clone https://github.com/Sdogruyol/cryload.git && cd cryload
```

## Usage

```
./cryload http://wwww.example.com
```
