# cryload

HTTP benchmarking tool writen in [Crystal](http://crystal-lang.org/)

## Installation

```
git clone https://github.com/Sdogruyol/cryload.git && cd cryload
crystal build src/cryload.cr --release
```

## Usage

```
./cryload http://wwww.example.com
```
